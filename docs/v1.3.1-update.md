# v1.3.1 — Live Preview, Collapsible Sidebar & Editor Improvements

**Date:** March 5, 2026

---

## Summary

This update adds a live preview panel to the page editor, makes the admin sidebar collapsible with icon navigation, and improves the rich text editor with scrollable max-height. The page editor now features a resizable split-pane layout with real-time preview and a "View Live" link.

---

## Changes

### 1. Live Preview Panel in Page Editor

**File:** `components/admin/page-preview.tsx` *(new)*

A client-side preview component that renders the page using the same templates as the public site. Receives the current `pageType`, `content`, and `fullPath` as props, looks up the registered template via `getTemplate()`, and renders it in real time as content is edited.

The preview panel includes:
- A header bar showing "Preview" label and the page's full path
- Scrollable content area with the rendered template
- Graceful fallback when no template is found for the page type

**File:** `app/admin/pages/[id]/page.tsx`

The edit page now features a split-pane layout:
- **Editor on the left** — The form with status, content properties, and action buttons, inside a scrollable card
- **Preview on the right** — Live-rendered preview using the page's registered template
- **Draggable divider** — A vertical drag handle between panels allows resizing the split (clamped 20%–80%)
- **Toggle button** — "Show Preview" / "Hide Preview" button in the page header toggles the preview panel on/off
- **View Live link** — An external-link button in the header opens the page's public URL in a new tab

When preview is hidden, the editor reverts to a single-column `max-w-2xl` layout.

### 2. Collapsible Admin Sidebar

**File:** `components/admin/admin-shell.tsx`

The admin sidebar is now collapsible:
- **Toggle button** — A chevron icon button in the sidebar header collapses/expands the sidebar
- **Collapsed state** — Sidebar shrinks from `w-64` to `w-14`, showing SVG icons instead of text labels
- **Nav icons** — A `navIcons` record maps each route to a distinct SVG icon:
  - Dashboard → grid icon
  - Pages → document icon
  - Files → folder icon
  - Page Types → layers icon
- **Tooltips** — Nav items show the label as a `title` tooltip when collapsed
- **Logout** — Collapsed mode shows an icon-only logout button; expanded mode shows the full user info and "Sign out" text
- **Animation** — Smooth `transition-all duration-200` on sidebar width change

### 3. Scrollable Rich Text Editor

**File:** `components/admin/rich-text-editor.tsx`

The TipTap editor and raw HTML textarea now have a max-height cap:
- Max height is calculated as `rows × 3rem` (e.g., `rows={8}` → `24rem` max)
- Content scrolls within the editor when it exceeds the max height
- The editor still expands up to the max height via its existing `min-height`
- Applies to both the visual WYSIWYG editor and the raw HTML editing mode

### 4. Documentation Updates

- **`README.md`** — Added "Page Editor with Live Preview" section, updated Admin Panel description with collapsible sidebar, noted scrollable max-height on the rich text editor, added page-preview to project structure
- **`.github/copilot-instructions.md`** — Added Admin Shell and Page Editor Preview sections, updated components listing, updated template docs for client-side rendering compatibility, updated inline styles rule for dynamic values
